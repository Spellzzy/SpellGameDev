# 阶段5：内容与资源 Prompt 模板

## 5.1 数值设计

```
我需要设计【系统名称】的数值体系。

类型：【成长曲线/战斗公式/经济系统/掉落概率/...】
参考：【有没有参考的游戏或风格偏好】
关键参数：
- 【参数1：含义和大致范围】
- 【参数2：含义和大致范围】

请帮我：
1. 给出核心公式
2. 用表格列出各等级/阶段的数值
3. 画出成长曲线的趋势（文字描述）
4. 标注需要策划调参的变量
5. 给出对应的 C# 数据结构和配置文件模板
```

## 5.2 PCG 算法设计

```
我需要一个程序化生成【地图/关卡/地形/物品/...】的算法。

要求：
- 生成规模：【大致尺寸/数量】
- 风格约束：【规则描述】
- 性能约束：【生成时间要求】
- 可控性：【哪些参数需要开放给策划调】

请先给出伪代码方案，确认思路后再写实现。
```

## 5.3 资源规格定义

```
我需要为【模块名称】定义资源规格。

资源类型：【UI图标/角色模型/场景贴图/特效/音频/...】
目标平台：PC + 手机

请给出：
1. 推荐的格式和尺寸
2. 命名规范
3. 目录组织方式
4. 压缩/图集设置建议
5. 内存预算估算
```
