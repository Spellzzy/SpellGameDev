# 阶段6：知识沉淀 Prompt 模板

## 6.1 功能完成后总结

```
我刚完成了【功能名称】的开发。

请帮我整理：
1. 功能概述（一句话）
2. 涉及的文件清单
3. 核心设计决策（为什么这样做，考虑过哪些方案）
4. 已知限制和后续可优化的点
5. 使用示例代码
```

## 6.2 架构决策记录 (ADR)

```
请帮我记录一条架构决策：

标题：【决策主题】
背景：【为什么要做这个决策】
考虑的方案：
- 方案A：【描述 + 优缺点】
- 方案B：【描述 + 优缺点】
最终选择：【选了哪个】
原因：【为什么选这个】
后果：【这个选择带来的影响】
```

## 6.3 阶段性项目回顾

```
请帮我回顾当前项目状态：

1. 已完成的模块清单和完成度
2. 进行中的工作
3. 待解决的技术债
4. 下一步优先要做的事
5. 风险提示
```

## 6.4 代码知识提取

```
请阅读以下代码/模块，帮我提取关键知识：

---
【粘贴代码或指定文件路径】
---

提取：
1. 这段代码解决什么问题
2. 核心算法/设计模式是什么
3. 关键的坑和注意事项
4. 可复用的部分
```
